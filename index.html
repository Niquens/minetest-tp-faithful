This texture pack provides 32x32px textures similar to the default textures from minetest.

Download

Disclaimer: This is a Work-In-Progress (WIP). feel free to fork+pull-request if you want to make changes or add textures.

The texture pack is available through git: 

git clone git://github.com/Zeg9/minetest-tp-faithful

You can also download it as a .zip or a .tar.gz, but you have to download it again for each update.

If you wish to contribute, follow this link to the github repository.
